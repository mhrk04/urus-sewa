Details of assessment:
Develop a simple Java GUI program to manage a mini car rental system. The program should allow users
to add new rental cars, search for cars by brand or model, and display a list of all available cars. Test and
implement a Java program that fulfills these requirements using programming principles, methods, and
arrays or ArrayLists.
In completing this task, you are expected to:
1. Write a Java program that meets the specified requirements for the car rental management system.
2. Your program should include appropriate methods to handle adding new cars, searching for cars,
and displaying the catalogue / rental details.
3. Utilize arrays or ArrayLists in your program to store and manage the car inventory.
4. Ensure that your program is user-friendly and provides clear instructions for users to interact with
it.
Submission guidelines:
1. Program criteria:
a. Include sufficient comments in your program to explain the code.
b. Code must be written in well-indented format for easy readability.
c. The program should produce error-free, neat and nice professional looking output with
good user interface and is user friendly.
2. Prepare a written report with the following requirements
a. Cover page (Program Title, Group member details – standard cover page)
b. Table of content
c. Introduction: Brief description of your program (Objectives, scope, target user, etc.)
d. Program Features: Print screen and briefly describe the features (Add, Search, Display,
etc.) in the program.
e. Desribe the methods and arrays in your program.
f. Discuss any challenges or issues that your group encountered during the implementation
and how you overcame them.
g. Conclusion: Summarize your findings and any potential areas for improvement.
h. Provide appropriate references to support your analysis and recommendations. Use APA
citation format.
i. The report should be approximately 5 – 15 pages (excluding cover page, table of contents,
references and appendices) and use clear headings and subheadings to organise your
report.
j. Save the file as pdf file (Example: Group1.pdf).
k. Font style: Arial/Calibri and Courier New (for coding)
l. Font size: 11pt – 16pt
